# @kgpacks/backend

Fastify + SSE service (replacing the FastAPI backend): chat (POST and GET stream), search, graph, hybrid, and articles endpoints, with rate limiting.

> **Status: Phase 0 skeleton.** This package is a buildable placeholder with no
> business logic yet. Its responsibility above mirrors the Default Stack in
> [docs/PLAN.md](../../docs/PLAN.md); implementation lands in a later phase, gated
> by parity tests. See [docs/monorepo.md](../../docs/monorepo.md) for the workspace
> layout and conventions.
